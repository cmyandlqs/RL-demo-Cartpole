
## state
[小车位置, 小车速度, 杆子角度, 杆顶端的速度]

## action 
向左或者向右

## DQN伪代码
```
# 1. 初始化准备
初始化 经验回放池 ReplayBuffer(容量=10000)
初始化 评估网络 Q_net(输入=4, 输出=2)
初始化 目标网络 Target_net(输入=4, 输出=2)，并复制 Q_net 的初始权重
初始化 优化器 Optimizer (如 Adam，负责更新 Q_net 的参数)

# 2. 开始大循环（比如训练 500 局游戏）
for episode in range(500):
    state = env.reset() # 拿初始状态
    
    # 3. 每局游戏里的小循环（一直走到小车倒下）
    while not done:
        # --- [交互阶段] ---
        if 随机数 < epsilon: 
            action = 随机选一个动作 (0 或 1) # 探索未知
        else:
            action = Q_net(state).argmax() # 利用已知：选 Q 值最大的动作
            
        next_state, reward, done = env.step(action)
        
        # 把这一步的经历塞进经验池
        ReplayBuffer.push(state, action, reward, next_state, done)
        
        state = next_state # 状态步进
        
        # --- [学习阶段] ---
        if 经验池里的数据量 > Batch_Size (比如 64 条):
            # 1. 采样：从经验池随机抓取 64 条旧数据
            batch_s, batch_a, batch_r, batch_next_s, batch_done = ReplayBuffer.sample(64)
            
            # 2. 算现实预期（用不更新的目标网络）：
            # 找出下一个状态最大的 Q 值，如果 done 了就是 0
            max_next_q = Target_net(batch_next_s).max()
            target_Q = batch_r + gamma * max_next_q * (1 - batch_done) 
            
            # 3. 算我的预测（用正在训练的评估网络）：
            # 算出在这 64 个状态下，我【当年实际采取的动作】对应的 Q 值预测
            current_Q = Q_net(batch_s).gather(batch_a) 
            
            # 4. 算误差并反向传播
            loss = MSE_Loss(current_Q, target_Q)
            Optimizer.zero_grad()
            loss.backward()
            Optimizer.step() # 更新 Q_net 的权重
            
        # --- [目标网络同步阶段] ---
        if 当前总步数 % 100 == 0:
            Target_net.load_state_dict(Q_net) # 把评估网络的最新脑子复制给目标网络
```
